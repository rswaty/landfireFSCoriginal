Using LANDFIRE Products for FSC compliance

Chapter 2 Software and Datasets

You will need these LANDFIRE products:

  • Spatial datasets, clipped to your area(s) of interest
    • Biophysical Settings. This dataset will be used to get at “community habitat”, or where ecosystems could occur based on abiotic factors (e.g., soils, climate).
    • Succession classes characterizes structural classes on the landscape at the time the dataset represents (e.g., 2016 for LF Version 200).
    • Existing Vegetation Type maps NatureServe’s Ecological Systems (see descriptions here).
  • Non-spatial products
    • Biophysical Settings Descriptions which has information on natural disturbance regimes and succession class descriptions (also available here)
    • Reference Condition Table supplements the BpS descriptions with the “reference” percentages for each succession class, for each Biophysical Settings.
